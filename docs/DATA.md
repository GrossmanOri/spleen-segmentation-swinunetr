# Data

This project uses **Task09_Spleen** from the **Medical Segmentation Decathlon (MSD)** — public, de-identified abdominal CT with spleen segmentation labels (41 volumes; a 32/9 train/validation split was used here).

- Official site: http://medicaldecathlon.com/
- The dataset is released under **CC-BY-SA 4.0**. It is **not** redistributed in this repository — download it yourself from the official source.

After downloading, place the extracted `Task09_Spleen/` folder where the notebook's data path expects it (see the data cell at the top of the notebook), or update that path to point at your copy.
